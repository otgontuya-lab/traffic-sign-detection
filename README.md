# Онцгой цэгт суурилсан авто замын тэмдэг таних систем

## Танилцуулга

Энэхүү төсөл нь “Онцгой цэгт суурилсан авто замын тэмдэг таних программ” сэдэвт бакалаврын дипломын ажлын хэрэгжүүлэлтийн хэсэг юм.

Системийн үндсэн зорилго нь видео болон бодит камерын дүрсээс авто замын тэмдгийг бодит цагийн горимд илрүүлж, жолоочид анхааруулга өгөхөд оршино.

Энэхүү систем нь deep learning загвар ашиглахын оронд SIFT feature detection, FLANN matcher, RANSAC homography, object tracking зэрэг computer vision аргуудыг ашиглан хэрэгжүүлсэн.

Систем нь дараах боломжуудтай:

- Замын тэмдэг илрүүлэх
- Олон тэмдгийг зэрэг таних
- Тэмдэг хүртэлх ойролцоох зай тооцоолох
- Жолоочид анхааруулга харуулах
- Видео болон бодит камерын дүрс боловсруулах

## Бодит камер дээр ажиллуулах

Энэхүү төсөл нь бодит камерын дүрс дээр ажиллах боломжтой.

## Iriun Webcam ашиглах

Энэхүү систем нь Iriun Webcam ашиглан утасны камерыг бодит камер хэлбэрээр ашиглах боломжтой.

Ажиллуулахын тулд:

1. PC дээр Iriun Webcam программыг суулгана.
2. Утас дээр Iriun Webcam app суулгана.
3. Утас болон компьютерийг нэг Wi-Fi сүлжээнд холбоно.
4. Iriun app-аа асаана.
5. Дараа нь:

```bash
python c.py
```
командыг ажиллуулна.

Хэрэв камер нээгдэхгүй бол `camera.py` файл доторх:

```python
CAMERA_INDEX = 0
```

утгыг:

```python
CAMERA_INDEX = 1
CAMERA_INDEX = 2
```

гэж өөрчилж туршина.

## Харагдах үр дүн

Систем ажиллах үед:

- Видео эсвэл камерын frame бүрийг боловсруулна.
- Замын тэмдгийг автоматаар илрүүлнэ.
- Илэрсэн тэмдгийг bounding box ашиглан тэмдэглэнэ.
- Замын тэмдэг хүртэлх ойролцоох зайг тооцоолно.
- Дэлгэцийн дээд хэсэгт анхааруулга харуулна.

Мөн илэрсэн frame-үүд `detected_frames/` folder дотор хадгалагдана.
